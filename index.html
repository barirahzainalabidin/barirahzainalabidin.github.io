Rohlik Orders Forecasting Challenge

This project involves predicting grocery delivery orders at selected warehouses for the next 60 days using time series methods. In this competition, I achieved a ranking of 512th place. I employed two primary methods:

  • Random Forest Regression:
    • Filled missing values (numerics with 0 and text data with 'unknown')
    • Converted 'date' to datetime format
    • Extracted common date features
    • Encoded categorical columns
    • Defined features and target
    • Split the data into features and target
    • Split the data into training and testing sets
    • Trained a RandomForest model
    • Predicted on the test split
    • Evaluated the model using MAPE
    • Predicted on the actual test set
  • Hidden Markov Models:
    • Transformed data into time series format
    • Sliced time series data
    • Operated on time series data
    • Extracted statistics from time series data
    • Built Hidden Markov Models for sequential data
    • Analyzed market data using Hidden Markov Models
